# ValidFilename

Replace invalid characters in the filename from a string

## Installation

Add this line to your application's Gemfile:

    gem 'valid_filename'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install valid_filename

## Usage

    str.valid_filename

    ValidFilename.replace(str)
